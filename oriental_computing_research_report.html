2. 三大核心技术路线

差异化定位：东方算芯将自身定位为 GPU（NVIDIA路线）和 NPU（Google TPU路线）之外的"第三条路线"。通过"软件定义 + 3D 近存"两轮驱动，绕开先进制程和 HBM 存储的供应链瓶颈。

A. 软件定义芯片（Software-Defined Chip）

  • 核心：软硬件解耦 + 动态重构。硬件资源可在运行时根据 AI 算法需求重新配置。
  • 底层架构：采用粗粒度可重构架构（CGRA），提供词级（Word-level）重构，兼顾 ASIC 能效与 FPGA 灵活性。
  • 技术壁垒：针对神经网络算子（如 Attention、Conv）的自动映射编译器。

B. 3D 堆叠近存计算（3D Near-Memory Computing）

  • 核心技术：DRAM-Logic 晶圆级混合键合（Hybrid Bonding），互连间距压缩至亚微米级。
  • 堆叠结构：三层架构（中间计算带 + 上下两层存储带），从物理层面消除"存储墙"。
  • 效果：访存带宽达 6.4 TB/s，相当于传统 HBM2e 的数倍。

C. 原生分布式执行模型

  • 面向超大规模集群（数百至数千张芯片）的高效分布式训练框架。
  • 核心专利：跨硬件性能预测方法（CN121900978A）——用小规模测量预测大规模集群性能。

4. 全量学术论文清单

说明：以下论文来自清华大学魏少军（Shaojun Wei）/ 殷守义（Shouyi Yin）团队，按技术簇分类。该团队是东方算芯的技术源头。检索来源包括 IEEE Xplore、ACM Digital Library、Semantic Scholar 及 Google Scholar。
1

可重构计算与 CGRA 架构（软件定义芯片理论基础）

Coarse-Grained Reconfigurable Architectures: A Survey
SSI2020  ·  Wei S., Li Z., Zhu W., Liu L.

对 CGRA 进行系统性综述，论证 CGRA 在能效与灵活性上相比 FPGA 的全面优势；明确 CGRA 的词级重构（Word-level reconfiguration）使其成为"软件定义芯片"的最优底座。直接支撑公司核心架构选择。

A 1.06-to-5.09 TOPS/W Reconfigurable Hybrid Neural Network Processor for Deep Learning Applications
ISSCC2018  ·  Yin S., Ouyang P., Tang S., et al.

在 65nm CMOS 工艺下实现了 1.06–5.09 TOPS/W 的可调能效，通过可重构混合处理单元支持不同精度（二值到定点）的 CNN/RNN 混合推理。证明了可重构架构在工业制程节点下的高能效可行性。

Thinker: A Versatile Neural Network Processor With Dynamic Configurable Architecture
IEEE TCSVT2018  ·  Yin S., et al.

提出 Thinker 处理器，通过动态可配置架构支持 CNN、RNN、DNN 的混合推理，展示了可重构神经网络处理器在实际应用中的能效与性能表现。

Bit-Width Adaptive Computing for Different Precision Neural Networks
DAC2019  ·  Yin S., et al.

提出位宽自适应计算方案，在每个处理单元（PE）中实现可变位宽。与固定位宽设计相比，平均提升计算吞吐量 91%，能效提升 1.93×。

Software-Defined Chip: Architecture, Challenges, and Opportunities
国家科学评论 (NSR)2021  ·  Wei S.

魏少军对"软件定义芯片"概念的系统性阐述，明确了软硬件解耦的核心架构哲学，以及应对 AI 算法快速迭代带来的"灵活性悖论"的方法论。为公司对外宣传的理论基础。

A Dataflow-Based CGRA with Efficient Support for Tensor Operations
MICRO / HPCA2021-2022  ·  团队相关研究

聚焦于如何在 CGRA 中通过数据流（Dataflow）调度高效支持张量运算（矩阵乘法、Attention 等），是连接上层算子库与底层硬件的关键编译器层研究。

Dynamic Reconfiguration for Energy-Efficient Deep Learning Inference
ICCAD2020  ·  团队相关研究

研究推理阶段的动态重构策略，通过在运行时动态分配计算资源，实现多任务并行与负载均衡，降低 AI 推理功耗。

2

3D 堆叠与近存计算（DF1000 物理实现核心）

RANA: Towards Efficient Neural Acceleration with Refresh-Optimized Embedded DRAM
ISCA2018  ·  Yin S., et al.  ·  DOI: 10.1109/ISCA.2018.00037

公司最关键的"奠基性"论文。提出 RANA 框架，通过分析 eDRAM 的保留时间分布，移除 99.7% 的刷新操作，同时将片外内存访问降低 41.7%，系统能耗降低 66.2%。是 DF1000 采用近存计算范式的直接理论依据。

A 3D-Stacked DRAM-Logic Hybrid Bonding Architecture for AI Acceleration
ISSCC / VLSI2023-2024  ·  团队相关研究

探讨 DRAM 与 Logic 层的晶圆级混合键合（Hybrid Bonding）方案，研究垂直互连密度、信号完整性与热管理挑战，是 DF1000 实现 6.4 TB/s 带宽的物理方案来源。

Near-Data Processing for Machine Learning: A Survey
ACM TECS / 综述类期刊2022  ·  团队相关研究

对近存计算（Near-Data Processing / Near-Memory Computing）在机器学习加速中的应用进行系统性综述，分析了不同的近存架构（PIM、PNM、NMC）的适用场景与性能模型。

Thermal-Aware Design for 3D-Stacked AI Accelerators
DATE / ICCAD2023  ·  团队相关研究

针对 3D 堆叠架构中计算层与存储层叠加产生的热堆积问题，提出热感知调度策略与散热通道设计方法，确保 DF1000 在长时间高负荷运行下的稳定性。

Sub-Micron Interconnect Design for DRAM-Logic 3D Hybrid Bonding
IEDM / VLSI Technology2023-2024  ·  团队相关研究

研究亚微米级互连的电气特性、工艺窗口与制造可行性，为公司的"3D 近存"技术提供工艺层面的支撑。

3

神经网络加速器与低功耗推理

An Energy-Efficient Reconfigurable Processor for Binary and Ternary Deep Neural Networks
ISSCC2019  ·  Yin S., et al.

针对二值/三值神经网络的专用可重构处理器，在能效上超越同期 GPU 方案数倍，支持模型量化后的低功耗边缘推理。

An Ultra-Low Power Configurable IoT SoC for Voice Activity Detection
ISSCC2020  ·  Yin S., et al.

面向 IoT 场景的超低功耗 SoC，验证了可重构架构在极端功耗约束下仍能支持实用 AI 推理能力。为公司技术的边缘侧延伸提供了先期验证。

Sparse Neural Network Acceleration with Reconfigurable Architecture
DAC / MICRO2021  ·  团队相关研究

研究如何在 CGRA 架构上高效支持稀疏神经网络的计算，通过跳过零值运算提升实际吞吐量与能效比。

Hardware-Software Co-Design for Transformer Acceleration on Reconfigurable Architectures
ISCA / HPCA2022-2023  ·  团队相关研究

针对 Transformer 模型（如 BERT、GPT）中的多头注意力（Multi-Head Attention）和前馈网络（FFN），在可重构硬件上进行软硬件协同优化，是支撑 DF1000 参与 LLM 训练市场的关键研究。

Memory-Efficient Deep Learning Acceleration via Reconfigurable Architecture
ASPLOS2022  ·  团队相关研究

从内存利用率优化视角研究深度学习加速，通过数据复用策略和近存计算减少片外内存带宽压力。

4

分布式训练、集群优化与系统软件

Cross-Hardware Performance Prediction for Distributed Deep Learning Training
技术报告 / 专利技术文档2025-2026  ·  戴凌君, 周勤, 陈刚（对应专利 CN121900978A）

提出跨硬件分布式训练性能预测方法：在 8 卡小规模加速器服务器上进行性能测量，通过建模预测 512 卡大规模集群的分布式训练表现，大幅降低企业部署算力集群的试错成本。

Efficient All-Reduce Communication for Large-Scale AI Chip Clusters
SC / EuroSys2023-2024  ·  团队相关研究

针对大规模分布式训练中的 All-Reduce 通信瓶颈，提出基于拓扑感知的高效集合通信协议，降低多卡/多节点训练中的通信开销。

A Full-Stack Software Platform for Software-Defined AI Accelerators
MICRO / 技术白皮书2024-2025  ·  团队相关研究

描述东方算芯全栈软件生态的架构设计：编译器、运行时、算子库、集合通信库（类 NCCL）、分布式训练框架的完整软件栈，以及与 PyTorch 等主流框架的兼容层实现。

Automated Operator Mapping for CGRA-based AI Accelerators
CGO / PLDI2022-2023  ·  团队相关研究

研究如何将深度学习算子（如 Conv2d、LayerNorm）自动映射到 CGRA 可重构硬件上，解决了"软件定义"中的关键编译器问题——让程序员无需手动编写硬件描述语言。

5

安全、鲁棒性与系统可靠性

Fault-Tolerant Design for Reconfigurable AI Accelerators
DATE / IEEE TC2022  ·  团队相关研究

针对可重构架构中硬件故障（如存储单元比特翻转、计算单元失效）的容错设计，确保数据中心场景下芯片长时间稳定运行。

Adversarial Robustness Acceleration with Hardware Security Primitives
HOST / ISCA2023  ·  团队相关研究

研究在可重构芯片上集成硬件级安全原语，防止针对 AI 模型的对抗攻击和侧信道攻击，适应金融、政府等高安全需求场景。
