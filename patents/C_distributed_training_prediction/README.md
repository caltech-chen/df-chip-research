## 技术领域

**核心技术：** 大模型分布式训练策略优化（跨硬件性能预测）

本专利解决的关键问题是：在大规模集群（如512张加速卡）上部署大模型训练时，如何在
不进行昂贵的真机测试的前提下，快速评估并筛选最优的并行训练策略。

**主要技术贡献：**
- 利用小规模/低成本已知硬件的实测性能数据，通过物理建模或神经网络方法预测目标硬件的训练性能
- 支持张量并行（TP）、流水线并行（PP）、数据并行（DP）、专家并行（EP）等多种并行策略的跨硬件性能迁移
- 基于 TGS（Tokens/GPU/s）和气泡比率（Bubble Ratio）对候选策略进行定量优先级排序
- 将策略优化时间从"数月真机测试"压缩至快速预测评估

**应用场景：** AI芯片/加速器选型与替换、大模型训练基础设施规划、训练策略自动搜索
