## 技术领域

**核心技术：** 芯片集成 / SoC设计模块层级动态管理（EDA工具领域）

本专利解决大型SoC芯片集成中的模块层级重组问题：当设计迭代需要将某个模块从当前
层级移动到另一个模块下时，传统方式需要从头重新集成整个设计，代价极高。

**主要技术贡献：**
- 采用树形（二叉树/链表）数据结构记录芯片所有模块的父子层级关系
- 通过寻找最近公共祖先（LCA）算法，实现任意两模块间的高效层级重组，复杂度为 O(depth)
- 支持模块"向上移动至公共祖先 → 再向下移动至目标位置"的两阶段原子操作
- 支持信号穿透（pass-through）模块的动态插入和信号路径的自动修改
- 自动更新集成代码（Verilog/SystemVerilog/RTL），减少人工编码错误

**应用场景：** SoC/ASIC设计集成、EDA自动化工具、芯片设计后端流程优化

> 注意：本目录命名为 `A_3d_interconnect`，但收录专利的实际技术方向为芯片集成方法
> （EDA/SoC模块层级管理），与3D互连技术不直接相关，请注意区分。
